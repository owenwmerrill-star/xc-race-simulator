# XC Race Simulator

A Python project to collect, analyze, and predict high school cross country race results.

## Goals

- Scrape race results from Athletic.net
- Build a clean database of runners, races, and courses
- Estimate runner fitness and course difficulty
- Predict race outcomes
- Build a web app for simulations
